
# Theme

**Light** and **dark** themes are available; you can also **follow the system** appearance.

## Acrylic effect

The acrylic (glass) effect is **off** by default. To enable it, open **Settings** → **General** and turn on **Enable acrylic effect**.

<img src="/images/screenshots/enable_acrylic.png" width="600" alt="Enable acrylic effect in settings" />

## Terminal themes

You can choose from several color themes for text rendering inside the terminal.
