# FaiTerm - 未来 AI 终端

FaiTerm 是一款面向未来的跨平台 AI 终端应用，集成本地与远程终端、文件传输、AI 助手等多项功能，助你高效管理服务器与开发环境。

> 🚀 **支持平台**：macOS / Windows / Linux

## 主要功能

- **本地终端**：原生终端体验，支持多标签页。
- **SSH 远程连接**：安全高效，支持密钥与密码登录。
- **SFTP 文件管理**：可视化文件操作，支持拖拽上传下载。
- **sz/rz 文件传输**：大文件极速传输，支持断点续传。
- **AI 智能助手**：命令自动补全，AI 问答，提升效率。
- **多平台支持**：一套应用，三大平台无缝切换。
